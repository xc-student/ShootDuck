水枪大战游戏 - 安装和运行说明
============================

系统要求：
--------
- 操作系统：Windows 7/8/10/11
- Java运行环境：Java 8或更高版本
- 显示器分辨率：最低800x600，推荐1280x720或更高

安装步骤：
--------
1. 确保您的电脑已安装Java运行环境（JRE）
   - 如果没有安装，请从 https://www.java.com/download/ 下载并安装
   
2. 解压游戏文件
   - 将所有文件解压到任意文件夹
   - 确保assets文件夹与game.jar在同一目录下

运行游戏：
--------
1. 双击start.bat文件启动游戏
2. 如果无法启动，请确保：
   - 已正确安装Java
   - assets文件夹存在且完整
   - 所有文件都在同一目录下

游戏控制：
--------
- 方向键：移动角色
- 空格键：发射水枪
- ESC键：暂停游戏/返回菜单
- F11键：切换全屏/窗口模式
- F10键：切换窗口大小

常见问题：
--------
1. 如果游戏无法启动：
   - 检查是否安装了Java
   - 使用命令行运行：java -jar game.jar
   - 查看是否有错误信息

2. 如果游戏运行缓慢：
   - 关闭其他占用资源的程序
   - 使用F10键降低分辨率
   - 更新Java到最新版本

3. 如果遇到其他问题：
   - 请确保所有文件完整
   - 检查assets文件夹是否完整
   - 重新解压游戏文件

祝您游戏愉快！
